Method   Endpoint                                                Params                              Desc
POST     /api/signup                                             email, password, (captcha)	         Create an account
POST     /api/login                                              email, password                     Login to an account
POST     /api/logout                                             csrf_token                          Log out

GET      /api/status/                                            -                                   Return status information (csrf_token, number of unread, ...)

GET      /api/feed/{feed_args/}                                  -                                   Returns feed with entries, formatted according to feed_args if present.
POST     /api/feed/id:<fid>/                                     -                                   Change feed settings or unsubscribe.

GET      /api/entry/id:<eid>/                                    -                                   Returns data for entry
POST     /api/entry/id:<eid>/                                    csrf_token                          Set unread or starred state.

POST     /api/subscribe                                          csrf_token, url                     Subscribe to a feed (by url), returns id for feed

GET      /api/settings/                                          -                                   Returns all settings (including values)
POST     /api/settings/                                          csrf_token, settings                Sets the settings to their new values



feed_args = "id:" , id , { id } |                           # Only return entries for feed(s) with supplied ids
            "tag:" , tag , "," { tag } |                    # Only return entries for feed(s) with supplied tags
            "filter:", filter |                             # Only return entries for feed(s) matching filters (may be removed/replaced/changed)
            "group:", ("feed" | "tag" | "none")             # Group returned entries by feeds, tags, or return all entries in a flat list (default: none)




Example return values:

    {"csrf_token": "dbInXNPd4fmbAeP15ri8wgcbm1g0R9GvSZnnG2Ddf2snVeoytv2uzZsSne6IdSKX+YbTQRTErREu", "unread": 0}
